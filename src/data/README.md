# Data Directory

This folder contains structured data for the CV content, mind-map nodes, comments, and likes.

## Guidelines
- Store CV sections, bullet points, and expandable node data here (e.g., as JSON or JS modules).
- Store comment and like data models for the Your Thoughts page.
- Keep data files organized by section (e.g., professional-summary.js, experience.js) and by feature (comments.js, likes.js).
- Update MEMORY.md if the data structure changes significantly.
