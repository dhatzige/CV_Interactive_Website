# Components Directory

This folder contains all React components for the Dimitris Interactive CV project.

## Guidelines
- Each component should have a single responsibility.
- Keep each file < 300-400 lines.
- Reuse components where possible to avoid duplication.
- Document any complex component logic inline and in MEMORY.md if architectural.

## Planned Components
- Navbar (navigation between Home and Your Thoughts)
- CVSection, CVBullet (CV display and interactivity)
- MindMapNode (expandable mind-map nodes)
- YourThoughtsPage (social CV page)
- CommentSection, CommentInput (comments for CV sections)
- LikeSection, LikeDialog (likes for CV sections)
