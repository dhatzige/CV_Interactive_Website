# Interactive CV Website

An experimental interactive CV website where sections expand into mind-map style conversations. Click any bullet point to explore follow-up questions and answers.

**Context:** An early project from my vibecoding journey. The concept was to make CVs more engaging than static documents. It is rough around the edges and represents where I started, not where I am now. For current work, see [CV-EXTRACTOR](https://github.com/dhatzige/CV-EXTRACTOR) or [TalentSpottingAI](https://www.talentspottingai.com).

## Features

- Interactive mind-map expansion on each CV section
- Social layer: comment on and like specific sections
- Retro web aesthetic with modern React interactivity

## Stack

- **Framework:** React 18 (CRA)
- **Language:** JavaScript
- **Routing:** React Router v6

## Built with

Built using AI-assisted development as one of my first experiments in making the traditional CV format more interactive.
